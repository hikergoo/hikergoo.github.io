                # 获取软件使用码

                软件使用码见附录答题链接

                # 使用前配置

                # 配置ngrok

                # 方法一
                • 申请到sunny_ngrok反向代理ID，（大约花费2元）。详情阅读 申请ngrok账号和隧道IDopen in new window，请注意，端口号码选择为127.0.0.1:10086 。 因为预约软件程序api端口固定了10086端口。

                  隧道申请时填写的端口号，127.0.0.1:10086

                  注意，请复制箭头的地址链接，这个链接可以通过互联网访问您的电脑。
                • 下载好的软件中 windows_amd64> Sunny-Ngrok启动工具.bat> 运行文件，输入客户端ID
                • 运行Sunny-Ngrok启动工具.bat . 记录红框中的Forwarding网址，这是手机访问电脑的关键。
                • 测试一下是否电脑已经成功做好内网穿透了，打开QuickYMQ软件，软件打开后会自动建立api程序。打开浏览器，输入你申请的网址例如：http://ouba.free.idcfengye.com/api/send_sms_code?code=100 , 正常情况下应该会看到下图的结果。
                # 方法二 【推荐】
                • 使用国内的花生壳内网穿透软件，原理跟上面一样。详情参考花生壳官方客户端教程open in new window 和博客花生壳 > 内网穿透 > 设置HTTPS映射教程open in new window
                • 注意事项，一定选择使用https的，本地端口为127.0.0.1:10086。 配置好，打开QuickYMQ软件，然后网页访问 https://填写你的二级域名名称.zicp.fun/api/send_sms_code?code=测试测试open in new window
                • 要是可以的话，花生壳更推荐些。（如果还有不懂的，请阅读QuickYMQ软件中说明文件.docx）

                # 配置手机端软件

                • 下载手机端软件 smsForward , 用来识别匹配短信验证码，并转发到你的电脑中。smsForward官网2open in new window
                • 配置smsForward： 先设置权限（非常重要，权限配置方法open in new window ），再配置通道，再配置转发规则。 短信转发规则设置为：【.*?】您的预约验证码为.*?(体育馆)

                # 正式使用

                运行

                • Sunny-Ngrok启动工具.bat 或花生壳客户端，开启自定义映射开关
                • QuickYMQ_Win_CMD_v4.0.exe
