            # 软件介绍

            # 开发目的

            当前主要矛盾是人民群众日益增长的对体育运动的热情与体育场所资源无法满足人民群众需要，以及手指头戳鼠标的速度远远慢于计算机程序自动化的矛盾。因此只能暂时借助工具稍微解决一下这个问题。

            基于以上原因，故出基于安卓、linux、python的安卓Termux还有windows客户端（windows命令行经典版）供选择。

            【温馨提醒】 该软件仅提供学习交流的思路，承诺不保存任何敏感信息（毕竟这玩意很刑）。测试完毕请务必立即删除！谢谢配合！

            # 推荐版本

            安卓Termux版本 推荐 ，配置简单。

            # 更新顺序

            首先是termux-python版本，其次是windows命令行经典版，其他的更新时间可能会贼慢，敬请谅解。
